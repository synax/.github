# Synax

We design, build, and run cloud platforms and applications for Swiss
financial institutions.

End-to-end: infrastructure, software, security, and compliance, delivered
as a single managed service. Founded 2018, based in Basel.

## What we do

- **Regulated cloud platforms** on Azure, with multi-tenant AKS, hub-and-spoke
  networking, Zero Trust access, and full audit trails.
- **Mobile and web applications** built with Flutter and Nuxt, observed
  end-to-end with Datadog.
- **Custom Kubernetes tooling**, operators, and infrastructure-as-code
  packages in Go and TypeScript on top of Pulumi, Flux, and Kpt.

## Compliance

ISO 27001, ISO 27018, ISO 27019, SoC 2 Type 2.
